
# Todo
basic todo project for teaching purpose only 

# Requirements
<ul>
 <li>maven comfigured to be able to run mvn</li>
  <li> Java 11</li>
</ul>

# Run unit tests
```sh
mvn test
```

# Run the microservice
 --- 
```sh
mvn spring-boot:run
```

## Acces to endpoints
http://localhost:8080/cicd/api/todos
